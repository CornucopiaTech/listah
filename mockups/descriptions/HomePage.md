# Workflows - Signed in - Visiting Home Page


- 2-grid system shows all category names and the number of items that fall under those categories.
- The page also includes the ability to add a new item. No item should be updated from this screeen.
- The navbar should have icons for going back to the home page, adding new item, filtering for item, and viewing their authentication profile.
- After the navbar, saved views (i.e. saved filters of tags and categories) should appear on the top of the page.
- If a saved filters exists or not, give the ability to add new saved filter.
- After the row that shows the saved filter, display a 2-column grid that shows all categories (with buttons that show the number of items that fall under the category) and all tags.
- At the base of the screen, have an add item and create filter floating button that can be clicked at anytime to create filters.
